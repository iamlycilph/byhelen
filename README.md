# Словарь · квиз по урокам с Еленой

## Состав
- `index.html` + `data.js` + `icon.png` — приложение (деплой-комплект)
- `app-standalone.html` — то же одним файлом, открыть и попробовать сразу
- `convert.py` + `overrides.json` — конвейер обновления словаря
- `report.md` — что переведено автоматически, что исключено, что пропущено
- `tests/` + `run_checks.sh` — автопроверки (тот самый «луп» на будущее)

## Попробовать сейчас
Открыть `app-standalone.html` в браузере.

## Выложить по ссылке (GitHub Pages)
1. В репозиторий положить три файла: `index.html`, `data.js`, `icon.png`.
2. Settings → Pages → deploy from branch.
3. Ссылка вида `https://<ник>.github.io/<репо>/`.

## На экран айфона
Открыть ссылку в Safari → «Поделиться» → «На экран “Домой”».
Иконка и полноэкранный режим уже настроены в `index.html`.

## Обновить словарь (главный сценарий)
1. Получить от Елены свежий `Andy.docx`.
2. `./run_checks.sh Andy.docx` — конвертация + все проверки.
   Если появились слова без перевода — скрипт покажет их список:
   добавить перевод в `overrides.json` и запустить ещё раз.
3. Закоммитить новый `data.js`. Всё.
Правки существующих переводов — тоже через `overrides.json`
(ключ = английская строка). Значение `""` исключает карточку,
объект `{"en":..., "ru":...}` правит и заголовок.

## Заметки и звёзды
Живут в localStorage браузера на устройстве — обновление `data.js`
их не трогает (ключ — от английского слова, не от позиции).
Между устройствами не синхронизируются: внизу приложения есть
«Экспорт заметок» / «Импорт».

## Что проверить руками
- В словаре фильтр «Авто-переводы» — 154 перевода добавлены не Еленой,
  помечены жёлтым бейджем «авто». Пройтись с Еленой.
- `report.md`, раздел «Пропущенные строки» — грамматика и примеры,
  которые парсер сознательно не взял. Убедиться, что там нет нужного.
